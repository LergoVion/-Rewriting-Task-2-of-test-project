Configuring RDP Control Settings

Purpose:

RDP Control enables you to configure Remote Desktop Protocol (RDP) settings and display window parameters. Options default and rdp5_only are set by default but you can create custom settings profiles, which you can rename for identification later.

Configuring RDP Control Settings

1. Navigate to Main menu > RDP Control > Settings .

2. To create a new custom settings profile, click + (insert image).

|TIP: Click (insert image) to close a settings profile tab.

3. To name the settings profile, enter information into the red empty field red rectangular empty field (INSERT IMAGE).

|TIP: Use informative names, for example, office_secure_users_settings_1.

4. Configure the settings below:

  • 4.1 To set idle length time, enter a value larger than 1 into Idle timeout in milliseconds.

    • |CAUTION: To avoid data loss, note that the Idle timeout value is in milliseconds. To access online help with the exact formula, click XX here XX .

  • 4.2 To set RDP display window maximum width, enter enter a value larger than 1 into Maximum display width in pixels.

  • 4.3 To set RDP display window maximum height, enter a value larger than 1 into Maximum display height in pixels.

  • 4.4 To set color depth for the RDP display window, enter the suitable value into Maximum display depth in bit per pixels.

    • |TIP: Click XX here XX for information about color depth values.

  • 4.5 To select the RDP version you want to allow, select either or more of the Enable RDP 4 | Enable RDP 5 | Enable RDP 6 or newer options.

  • 4.6 To avoid potential compatibility problems, select Enable RDP 4-style authentication.

  • 4.7 To allow an evaluation before the connection is established, select Enable pre channel check.

  • 4.7 To reduce the risk of network overload during RDP sessions, select Enable compression.

    • |CAUTION: Failing to allow compression may result in network overload by up to 500%.

  • 4.8 To change the default -- - AUTO suffix, enter the approppriate suffix into the Autologon domain suffix.

  • To save the settings profile, click [COMMIT]. You can create more of these custom settings profiles by repeating the steps above.

    • |TIP: You can later select the custom settings profile you wish to use by navigating to Main menu > RDP Control > Connections.
